## Уточнённые макеты кнопок (1–9)

### Логика сетки

```
1 кнопка  → 1 колонка, полная ширина
2 кнопки  → 2 строки по 1 кнопке (вертикально, полная ширина)
3 кнопки  → 1 пара + 1 широкая
4 кнопки  → 2 пары
5 кнопок  → 2 пары + 1 широкая
6 кнопок  → 3 пары
7 кнопок  → 3 пары + 1 широкая
8 кнопок  → 4 пары
9 кнопок  → 4 пары + 1 широкая
```

### CSS-реализация (Vue/React компонент)

```css
.buttons-grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 12px;
  padding: 12px;
  height: 100%;
  /* Кнопки равномерно заполняют экран по высоте */
  grid-auto-rows: 1fr;
}

/* Последняя кнопка при нечётном количестве */
.relay-button:last-child:nth-child(odd) {
  grid-column: 1 / -1;   /* растянуть на обе колонки */
}

/* 1 кнопка — тоже на всю ширину */
.relay-button:only-child {
  grid-column: 1 / -1;
}
```

```jsx
// RelayButton.vue / RelayButton.jsx
// Минимальная высота кнопки чтобы не было мельче 72px
// при 9 кнопках на экране 844px (iPhone 14):
// (844 - 12*2 - 11*12) / 5 строк ≈ 137px — комфортно
```

### Высота кнопок при разном количестве

| Кнопок | Строк | Высота кнопки (≈iPhone 14, 844px) |
|--------|-------|-----------------------------------|
| 1 | 1 | ~820px |
| 2 | 2 | ~400px |
| 3–4 | 2 | ~400px |
| 5–6 | 3 | ~260px |
| 7–8 | 4 | ~190px |
| 9 | 5 | ~148px |

Все варианты укладываются в экран без скролла и остаются удобными для нажатия пальцем.
